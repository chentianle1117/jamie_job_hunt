## Troubleshooting

| Problem | Fix |
|---------|-----|
| `run_oracle.ps1` syntax error | Check encoding — file must be saved as UTF-8 with BOM. Re-save from Cowork if needed. |
| Email not sending | Check `gog.exe` is at `C:\Users\chent\go\bin\gog.exe` and Gmail account is authorized |
| Telegram not sending | Verify bot token and chat ID. Try `https://api.telegram.org/bot{TOKEN}/getMe` in browser |
| Notion entries not appearing | Check Notion DB URL. Verify the `data_source_id` matches `442438a9-e372-48b7-b5f5-5f6ed8ee8e99` |
| LinkedIn/job boards blocked | Expected — Cowork VM blocks job board domains. Claude uses WebSearch instead |
