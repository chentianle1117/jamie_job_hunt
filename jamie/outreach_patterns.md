## Template C: LinkedIn Connection Note (250 char limit)

For L&D roles:
```
Hi [Name] — I'm a Program Manager with an HR & L&D background exploring [Company]'s 
work in people development. Would love to connect and learn more about the team.
```

For EX / engagement roles:
```
Hi [Name] — I'm a data-driven People Ops PM interested in [Company]'s employee 
experience work. Would love to connect — happy to share background if helpful.
```

**Rules:**
- Under 250 characters
- No "I saw you work at..." opener
- One specific signal that you know what they do
- End with a low-friction ask

those skills.

Jamie (Yi-Chieh) Cheng

## Do / Don't Quick Reference

| DO | DON'T |
|---|---|
| Open with the reframe hook for formal outreach | Open with "I am passionate about..." |
| Use em dash for contrast / elaboration | Use "leverage" as a standalone buzzword |
| Cite specific numbers ($340K, 78%, 17 launches) | Say "perfect fit" or "dream company" |
| Propose a specific meeting time | Leave a vague "let me know" CTA |
| Use "Sincerely" for formal / first contact | Use "Warmly" (too intimate for cold) |
| Use "Best" for recruiter replies / warm emails | Use "Regards" (too cold/corporate) |
| Group skills in tricolons (X, Y, and Z) | Use more than one bullet list per email |
| Customize the company-specific line | Send identical body to every company |
| Mirror formality of the person writing to you | Default to maximum formality always |
