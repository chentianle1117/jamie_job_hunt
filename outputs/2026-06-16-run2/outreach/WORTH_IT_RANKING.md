## Before sending any of these

1. Verify the recipient STILL works at the company via live LinkedIn (current role, current employer).
2. Find a first name — "Hi [Hiring Manager]" in an email is a missed opportunity.
3. Check if ElevenLabs has H-1B filing history for non-tech roles — if yes, the sponsorship
   disclosure in the email is worth including. If unknown, include it anyway (screens efficiently).
4. LinkedIn connection notes: count characters before sending — the 300-char limit is strict.
