# Tailored Resumes — Where Finished Applications Live

**This folder = one subfolder-free folder of completed, application-ready resumes.**
Each application gets two files: an HTML (with change highlights) and a PDF (the file you submit).

---

## How to find the right file

- **File to submit:** `Company_RoleType_YYYY-MM-DD.pdf`
- **File to review changes:** `Company_RoleType_YYYY-MM-DD.html` (open in Chrome — has yellow/blue/red diff highlights)
- **Active working file** (what Claude is currently editing): `jamie/resume.html`
- **Latest exported PDF** (quick access): `jamie/resume_tailored.pdf`

## Starting a new application?

1. Tell Claude the job description
2. Claude picks a template from `resume_templates/` and copies it to `jamie/resume.html`
3. Claude tailors it — yellow/blue/red highlights show every change and why
4. Refresh Chrome to review: `open jamie/resume.html`
5. Say "export" → Claude generates the PDF and saves it here with the company name
6. Submit the PDF from this folder
